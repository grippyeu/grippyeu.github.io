Contenu de l'Audit :

Le logiciel Auditez propose un audit complet de compte publicitaire Facebook basé sur une compilation de framework leaders dans l'industrie de la performance publicitaire Meta Ads.

  • Structure : Budget, lookalike usage, CBO vs ABO repartition, 
  • Budget : % Spent in learning, funnel budget repartition
  • Targeting : Detailed Targeting Expansion, Automatic Placements, Exclusions Implementation ; Prospecting Audiences
  • Optimisation : Advertising Objectives ; Optimizing for purchases ; success measurement ; custom conversions 
  • Creatives : ad format checks, content sizing checks
  • Basis : pixel and domain configuration checks
